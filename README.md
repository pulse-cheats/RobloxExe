Scarlet is a popular iOS sideloading solution used for installing IPA applications directly on iPhone and iPad devices. It supports installing IPA files without requiring a jailbreak. ￼

📥 Installation
	1.	Download the IPA.
	2.	Install using Scarlet or KSign.
	3.	Trust the certificate in iOS Settings.
	4.	Launch the application.
	5.	Enjoy.
